# Installation
```
npm install rorop -g
```
# Usage

bundle js plugin with [rollup](https://rollupjs.org/guide/en)
# Command
### 1. rop init

```
rop init xscroll
```
### 2. before bundle
```
cd xscroll
/* 
 * coding in ./src/main.js (default entry)
 */
 
```
### 3. rop build
```
rop build
```
